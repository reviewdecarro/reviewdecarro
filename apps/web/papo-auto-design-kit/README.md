DIRETRIZES DE USO:
-----------------

✓ FAZER:
  • Usar versão colorida (#C8481B) sobre fundos claros
  • Manter proporções originais do logo sempre
  • Garantir área de respiro mínima igual à altura do símbolo
  • Usar versão branca sobre fundos escuros ou fotos
  • Aplicar apenas sobre fundos de alto contraste

✗ EVITAR:
  • Distorcer, esticar ou rotacionar o logo
  • Alterar as cores da paleta estabelecida
  • Adicionar efeitos, sombras ou gradientes ao logo
  • Usar sobre fundos de baixo contraste
  • Modificar espaçamento entre símbolo e texto
  • Recriar ou redesenhar elementos do logo

TIPOGRAFIA:
----------

Fonte: System UI
Stack: -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif

Pesos:
  • Black (900) - Títulos principais
  • Bold (700) - Subtítulos
  • Semibold (600) - Destaques
  • Regular (400) - Corpo de texto

ÁREA DE PROTEÇÃO:
----------------

Mantenha uma área livre ao redor do logo equivalente à altura
do símbolo em todos os lados.

FORMATO DOS ARQUIVOS:
--------------------

Todos os arquivos são fornecidos em formato SVG (Scalable Vector Graphics)
para garantir qualidade perfeita em qualquer tamanho.

SUPORTE:
-------

Para dúvidas sobre o uso da marca, entre em contato com a
equipe de design da Papo Auto.
